# Compile to OpenAI Agents (end-to-end)

This example compiles the `SKILL.md` + `effector.toml` in this folder into an OpenAI Agents `FunctionTool` definition using `@effectorhq/core`.

## Run

If you run this example outside the `effector-core` repo, run `npm install @effectorhq/core` first.

```bash
node compile.js
```

## Output

- `output/openai-agents-function-tool.json`
