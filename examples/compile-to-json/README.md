# Compile to JSON (end-to-end)

This example compiles the `SKILL.md` + `effector.toml` in this folder into raw Effector IR using the `json` compile target.

## Run

If you run this example outside the `effector-core` repo, run `npm install @effectorhq/core` first.

```bash
node compile.js
```

## Output

- `output/effector-ir.json`
