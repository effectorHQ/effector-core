# Compile to MCP (end-to-end)

This example compiles the `SKILL.md` + `effector.toml` in this folder into an MCP tool schema using `@effectorhq/core`.

## Run

If you run this example outside the `effector-core` repo, run `npm install @effectorhq/core` first.

```bash
node compile.js
```

## Output

- `output/mcp-tool-schema.json`
